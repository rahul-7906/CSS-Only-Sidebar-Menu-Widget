A lightweight, modern, and completely responsive slide-out navigation sidebar built entirely with HTML5 and CSS3. 

## ⚡ The Challenge & Core Feature
The primary goal of this mini-project was to build an interactive UI component **without writing a single line of JavaScript**. 

Interactive open/close actions are managed cleanly using the **CSS Checkbox Hack** (`:checked` pseudo-class combined with sibling combinators) to handle state changes efficiently and natively within the browser styling pipeline.


## 🎨 Features
* **Zero JavaScript:** Full layout interaction managed purely through native HTML checkboxes and CSS triggers.
* **Smooth Animations:** Integrated CSS `transition` properties for slick, ease-in-out hover and sliding motion paths.
* **Polished UI Elements:** Custom modern typography loaded via Google Fonts (Poppins) and responsive scalable graphics powered by Font Awesome.
* **Mobile-First Asset Pipeline:** Developed entirely inside a mobile-based compilation setup!

## 🛠️ Built With
* **HTML5** - Layout structure and semantic components.
* **CSS3** - Custom styles, checkbox state-hacks, flex properties, and custom animations.
* **Google Fonts** - Poppins font integration.
* **Font Awesome v6.4.0** - Vector design elements and menu icons.
